# Line-Following-Robot
<h2>A simple line following robot powered by an Arduino Uno.</h2>


<h4>Parts list:</h4></br>
1 x Arduino Uno</br>
1 x L298N Dual H Bridge module</br>
2 x DC Geared motors (6.0 volts, 300 RPM)</br>
3 x IR Sensor</br>
robot chassis</br>
Caster wheel</br>
1 x 9v battery</br>
2 x wheels</br></br>
